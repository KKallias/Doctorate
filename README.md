# Medical Website Practice Project

A front-end practice project based on a medical-services landing page.

## Purpose

The project was created to practise translating a visual layout into a structured web page.

## Built with

- HTML
- CSS

## Current scope

The repository contains a single-page website with:

- Header and navigation
- Hero section
- Service information cards
- Supporting image and icon assets

This is a learning project; contact details, links, and appointment actions are placeholders.
